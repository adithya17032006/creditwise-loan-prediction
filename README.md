# CreditWise Loan Approval Prediction

## Overview
Machine learning project that predicts whether a loan application will be approved based on applicant financial and credit-related attributes.

## Features
- Data preprocessing
- Feature engineering
- Loan approval prediction
- Model evaluation

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib

## Dataset Features
- Credit Score
- Applicant Income
- DTI Ratio
- Loan Amount

## Model Used
- Gaussian Naive Bayes
